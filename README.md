# TripMate

A small web app that helps **a group of friends on a trip** split expenses and build a list of places to visit — no login, just share a link.

> Personal project, for me and my friends. Non-commercial.

## Features (MVP)
- 💸 **Split expenses**: record an expense (who paid, split among whom), even or exact split.
- 🧮 **Settle up**: see who owes whom, get a fewest-transfers suggestion, mark as paid.
- 📍 **Places**: a list of places to visit, mark them as visited.
- 🔗 **No login**: one trip = one link, open and go (optional per-trip PIN for privacy).

See [SPEC.md](./SPEC.md) for details.

## Tech stack
Next.js (App Router) · TypeScript · Supabase (Postgres) · Tailwind + shadcn/ui · Vercel
